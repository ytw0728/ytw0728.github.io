Key Features:

  • ⚡ Instant fuzzy search from address bar (type "/" to activate)
  • ⭐ Mark frequently used bookmarks as favorites
  • 📁 Choose custom root bookmark folder
  • ⌨️ Keyboard shortcuts for quick access
  • 🌐 Multi-language support (English, Korean)
  • 🔒 100% local - no data leaves your device

주요 기능:

  • ⚡ 주소창에서 즉시 퍼지 검색 ("/" 입력으로 활성화)
  • ⭐ 자주 사용하는 북마크를 즐겨찾기로 표시
  • 📁 커스텀 루트 북마크 폴더 선택
  • ⌨️ 빠른 접근을 위한 키보드 단축키
  • 🌐 다국어 지원 (영어, 한국어)
  • 🔒 100% 로컬 - 데이터가 기기를 벗어나지 않음
